记录Hexo安装过程

注：

检验=检查***是否安装成功，防止安装失败造成下一步报错

  1. 安装Git和Node.js。（选择对应的版本下载安装即可）

检验：在桌面右键，如果有Git Bash Here这一选项则表明Git安装成功。按下window+R按键，输入cmd，打开命令提示符窗口（下文简称打开cmd），输入：node -v返回版本号，输入npm -v返回版本号，则表示Node.js安装成功。

  1. 安装Hexo框架：

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

打开cmd，然后输入:

显示如下图所示，则表明安装完成。（因为国内网络环境较差，可能要等很久….）

检验：接着输入hexo -v如果输出一列版本号，则表示Hexo安装成功

  1. 使用Hexo框架生成网页：

在本地建立名为blog的文件夹（路径不要有中文），进入文件夹，右键选择Git Bash Here（下文简称打开Git），如下图所示。

输入hexo init（即：初始化hexo），若出现如下图的报错重试几遍就好了（我重试了6遍）

成功显示：

接着输入hexo s启动 hexo，在浏览器进入http://localhost:4000即可访问刚才生成的网页

  1. 将生成的网站部署到Github上

通过上面几步已经完成网站的初始生成，但只能在本地访问，如果想让别人看到就需要部署到GitHub上

进入GitHub注册或登录你的账号，如果注册如实填写你的信息即可

登陆后，进入控制台（主页面），点击右上角+号选择New repository新建一个仓库，在Repository name（储存器名称）一栏填你的昵称.github.io。例：我的昵称为ansizheshi，则填入ansizheshi.github.io

点击Create repository创建储存仓库。

接着回到3. 创建的blog的文件夹，进入文件夹，打开Git，输入npm install --save hexo-deployer-git，安装一个hexo部署插件。

配置git

在任意位置打开git，分别输入git config --global user.email "your@email.com"和git config --global user.name "your name"

{your@email.com处填上你的邮箱，your name处写上你的用户名（用户名可随便填）}

然后输入ssh-keygen -t rsa -C "your@email.com"（your@email.com处填上你的邮箱地址）
